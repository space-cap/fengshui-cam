# Product Requirements Document (PRD)
**프로젝트명:** AI 풍수지리 인테리어 테스트 (가칭: lucky-room)
**문서 버전:** 1.0.0
**작성일:** 2026-04-03

---

## 1. 프로젝트 개요 (Overview)
* **목적:** 복잡한 개발 과정 없이 빠르고 가볍게 완성하여 기분 전환 및 성취감을 얻기 위한 퀵(Quick) 프로젝트.
* **핵심 가치:** 최신 멀티모달 AI 기술을 활용하여, 사용자에게 '사진 한 장으로 보는 재미있는 풍수지리 분석'이라는 색다른 경험 제공.
* **타겟 사용자:** 자신의 방이나 자취방 인테리어에 관심이 있고, 가벼운 운세나 심리 테스트를 즐기는 모바일 웹 사용자.

## 2. 주요 기능 (Key Features)
### 2.1. 이미지 업로드 (모바일 최적화)
* 사용자가 스마트폰 카메라로 방을 촬영하거나 갤러리에서 사진을 선택하여 업로드.
* **제약 사항:** 업로드 파일 포맷은 이미지(`image/*`)로 제한.

### 2.2. AI 풍수지리 분석 및 로딩 처리
* 업로드된 이미지를 Base64 형태로 인코딩하여 OpenAI API로 전송.
* AI 처리 시간(약 3~5초) 동안 사용자 이탈을 막기 위한 직관적인 로딩 UI(스피너 및 텍스트) 제공.

### 2.3. 결과 출력
* AI가 분석한 다음의 세 가지 요소를 화면에 깔끔하게 렌더링:
  1. 전체 풍수 점수 (100점 만점)
  2. 재물운/건강운 상승을 위한 구체적인 인테리어 조언 2가지
* 줄바꿈 등의 텍스트 포맷을 유지하여 가독성 확보.

## 3. 사용자 플로우 (User Flow)
1. **[진입]** 메인 웹페이지 접속.
2. **[액션]** '사진 업로드' 영역 터치 -> 카메라 촬영 또는 사진 선택.
3. **[액션]** '운기 분석 시작하기' 버튼 클릭.
4. **[대기]** 로딩 스피너 및 "AI가 기운을 살피는 중입니다..." 메시지 노출 (이때 버튼은 비활성화).
5. **[결과]** 화면 전환 없이 하단에 AI 분석 결과(점수 및 조언) 출력.

## 4. 기술 스택 (Tech Stack)
* **Frontend:** HTML5, Vanilla JavaScript, Bootstrap 5 (빠른 UI 구현 및 반응형 레이아웃)
* **Backend:** PHP 8.x (단일 `index.php` 파일로 뷰와 서버 로직 통합 처리)
* **AI API:** OpenAI API (`gpt-4o` 모델) - Vision 기능 활용
* **통신 방식:** PHP `cURL`을 이용한 REST API 호출

## 5. 단계별 개발 계획 (Milestones)
* **Phase 1 (MVP - 당일 완성 목표):** * 단일 PHP 파일 기반 구현.
  * DB 연동 및 로그인 없이, 접속자 누구나 즉시 테스트 가능하도록 열어둠.
* **Phase 2 (향후 고도화 고려 사항 - Out of Scope for now):**
  * 분석된 사진과 결과를 DB에 저장하여 '나만의 기록' 생성.
  * 카카오톡/URL 공유 기능 추가.
  * AWS S3 등을 활용한 체계적인 이미지 스토리지 관리.
